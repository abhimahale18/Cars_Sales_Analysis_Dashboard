# 🚗 Cars Sales Analysis Dashboard | Power BI & SQL

## 📌 Project Overview

This project is an interactive **Cars Sales Analysis Dashboard** built using **Microsoft Power BI** and **SQL Server Management Studio (SSMS)**. The goal is to transform raw car sales data into meaningful business insights that support data-driven decision-making.

---

## 🎯 Objective

- Analyze overall car sales performance.
- Identify yearly sales trends.
- Compare vehicle sales by fuel type and transmission.
- Analyze model-wise sales distribution.
- Enable interactive filtering for detailed business analysis.

---

## 🛠️ Tech Stack

- Microsoft Power BI
- SQL Server Management Studio (SSMS)
- SQL
- Power Query
- DAX
- Data Cleaning & Transformation

---

## 📂 Project Workflow

### 1. Data Extraction & Querying (SQL - SSMS)
- Retrieved data using SQL queries.
- Filtered and prepared the dataset.
- Validated data quality before visualization.

### 2. Data Transformation (Power Query)
- Cleaned missing and inconsistent values.
- Corrected data types.
- Removed duplicates.
- Prepared data for reporting.

### 3. Data Modeling
- Built relationships between tables.
- Optimized the data model.

### 4. Dashboard Development
Created interactive visualizations including:
- KPI Cards
- Sales by Year
- Cars by Model
- Fuel Type Analysis
- Manual vs Automatic Comparison
- Interactive Slicers (City, Color, Model)

---

## 📈 Key Insights

- Sales reached their highest level in 2023.
- Sales declined in 2024, highlighting an opportunity for further business investigation.
- Diesel and Electric vehicles have strong customer demand.
- Manual and Automatic transmissions have nearly equal sales.
- Sales are distributed fairly evenly across different car models.

---

## 💡 Business Recommendations

- Investigate the causes of lower sales in 2024.
- Increase inventory for high-demand fuel types.
- Continue offering both Manual and Automatic variants.
- Use city-level analysis for targeted marketing campaigns.
- Monitor model performance to improve inventory planning.
